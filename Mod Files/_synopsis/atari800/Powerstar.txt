Powerstar
Platform: Atari 800
Region: USA
Media: Cartridge
Controller: Keyboard/Joystick
Genre: Adventure - Text - Graphic
Gametype: Licensed
Release Year: 1985
Developer: Pandora Software
Publisher: Pandora Software
Players: 1
_________________________

Unusual for a cart-based game, this is a graphic text adventure that supports both the keyboard and joystick. Supposedly, this game was never officially released and only 700 copies exist. It does however come in a box with instructions.
